Project Description
This project demonstrates how to build a responsive landing page with an interactive fixed navigation bar.

The navigation menu:

Remains fixed at the top of the page
Changes background color when the user scrolls
Includes hover effects on menu items
Adapts to mobile screens using a hamburger menu
Supports smooth scrolling to sections

The project is built using only:
HTML – Structure of the webpage
CSS – Styling and responsiveness
JavaScript – Scroll effects and interactivity

This project is ideal for beginners learning front-end web development.

Features

✅ Fixed navigation bar
✅ Scroll-based style change
✅ Hover effects on menu items
✅ Smooth scrolling
✅ Fully responsive design
✅ Mobile hamburger menu
✅ Clean and simple UI

🛠️ Technologies Used

HTML5

CSS3 (Flexbox & Media Queries)

JavaScript (DOM Manipulation & Event Handling)
